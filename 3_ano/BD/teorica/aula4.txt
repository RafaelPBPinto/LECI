	SQL
Linguagem para definir, manipular e questionar uma Base de Dados Relacional

Sublinguagem desta aula
DDL - Data Definition Languagem

	DDL
Permite definir várias entidades da BD

Utilizada para especificar a informação acerca de cada relação
	. esquema
	. domínio
	. Restrições de integridade

Tipos de dados básicos (podem variar de SGBD para SGBD)
	. Numbers
	. Characters, strings
	. Date e time
	. Binary objects

CHAR -> comprimento fixo
VARCHAR -> comprimento variável
NVARCHAR -> Árabe

Comando
Create domain -> permite definir novos tipos de dados
Create Type -> criar nvo alias
Default -> definir valor por omissão

Unique -> chaves candidatas não primárias

Definir ações alternativas para quando houver violação ao modificar foreign key
	on delete ou on update +
		. restrict - não deixa efetuar a operação
		. cascade - apaga os registos associados ou altera a chave estrangeira
		. set null 
		. set default

Alter - adicionar/eliminar atributos a tabela
